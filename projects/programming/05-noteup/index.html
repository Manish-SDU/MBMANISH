I present NoteUp, a lightweight web app designed to securely store account information locally on your computer in a user-friendly interface.

Originally a simple console-based tool I created to store trivial login data, NoteUp evolved from a basic terminal app into a visual, browser-based experience. I built it to explore web development more deeply, combining my growing knowledge of HTML, CSS, Python, and JavaScript.

The app allows you to log in securely and manage your saved credentials, all stored locally (when downloaded) to ensure your data stays private. It also supports Google login, random profile pictures, and password recovery using security questions.
